Материалы по лабораторным по ВЕБу (7 семестр) МГТУ им. Баумана ИУ7 

# Как запустить

## Env
Создать .env файл в папке `backend/src`, скопировав данные из env_sample

## Запуск

```bash
docker compose up -d
```
